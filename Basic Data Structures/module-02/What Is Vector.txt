What is Vector?
Vector is under STL or Standard Template Library. Vector is a Data Structure Library.

এখানে বিভিন্ন ধরনের ডাটা স্ট্রাকচার লাইব্রেরী ইমপ্লিমেন্ট করা আছে আমরা এগুলোকে ব্যবহার করে থাকি। আমরা যদি ডায়নামিক এ্যারে নিয়ে কাজ করতে চাই তাহলে সেই ডায়নামিক এ্যারের একটা লাইব্রেরী হলোঃ Vector.

Vector dynamic array নিয়ে কাজ করার প্রসেস গুলোকে আরো ইজি করে দেয়। আছাড়া আমরা চাইলে ডায়নামিক এ্যারের সাইজ বাড়াতে পারি।
